### **1. 核心功能模块与技术栈**
#### 📅 **基础功能**
| 模块              | 技术选项                                                                 | 推荐理由                                                                                   |
|-------------------|------------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| **GUI 界面**       | - **Tkinter**（Python内置）<br>- **PyQt5**（更美观）<br>- **PySimpleGUI**（简化版） | Tkinter 无需安装，适合快速开发；PyQt5 可展示复杂界面能力                                    |

### **1. 初始化数据库（database.py）**
```python
import sqlite3

class Database:
    def __init__(self, db_name="schedule.db"):
        self.conn = sqlite3.connect(db_name)
        self._create_table()

    def _create_table(self):
        # 创建任务表（包含最简字段）
        self.conn.execute('''
            CREATE TABLE IF NOT EXISTS tasks (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                title TEXT NOT NULL,
                due_time TEXT  -- 先用字符串存储，后期再转datetime
            )
        ''')
        self.conn.commit()

if __name__ == "__main__":
    # 测试数据库是否正常创建
    db = Database()
    print("数据库初始化完成！")
```

### **2. 创建基础窗口（main.py）**
```python
import tkinter as tk
from database import Database

class App:
    def __init__(self):
        self.db = Database()  # 连接数据库
        self.window = tk.Tk()
        self.window.title("我的日程管理")
        self._build_basic_ui()
        
    def _build_basic_ui(self):
        # 添加一个标签和按钮
        label = tk.Label(self.window, text="欢迎使用日程管理")
        label.pack(pady=10)
        
        test_btn = tk.Button(self.window, text="测试按钮", command=self._test_db)
        test_btn.pack()
        
    def _test_db(self):
        # 测试数据库连接
        self.db.conn.execute("INSERT INTO tasks (title, due_time) VALUES (?, ?)", 
                           ("测试任务", "2024-05-20 10:00"))
        self.db.conn.commit()
        print("测试数据已插入数据库！")

if __name__ == "__main__":
    app = App()
    app.window.mainloop()
```

### **常见问题解决方案**
| 问题现象                  | 解决方法                                                                 |
|--------------------------|------------------------------------------------------------------------|
| 报错 `No module named tkinter` | Linux系统需安装：`sudo apt-get install python3-tk`                    |
| 数据库文件未生成           | 检查文件路径是否有写权限，或尝试绝对路径：`sqlite3.connect("/完整路径/schedule.db")` |
| 按钮点击无反应             | 确保按钮的 `command` 参数指向正确的方法（如 `self._test_db` 不要加括号）           |
